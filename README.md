# ✨ About The Project

MSN Mart is a modern full-stack e-commerce platform developed using Spring Boot and Thymeleaf.  
The project is designed with enterprise-level architecture and integrates secure authentication, OTP verification, payment gateway systems, email services, and responsive UI design.

The application provides a complete shopping experience where users can:

- Browse products
- Add products to cart
- Place secure orders
- Make online payments
- Receive OTP verification
- Get email confirmations
- Manage orders securely
